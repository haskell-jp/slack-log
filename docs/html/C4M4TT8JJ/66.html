■teratailについて

1. コミュニティの独立したページの開設
2. コミュニティの運営メンバーとしてteratail内で活動
3. teratailユーザーへコミュニティの活動を「お知らせ」として発信
4. Slackなどの外部コミュニケーションツールで、コミュニティに関係する技術の質問をウォッチ
