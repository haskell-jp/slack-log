VS Code Inline REPL: 動きました　確かめたことは以下の通り：
VS Code の中で Toolbar ＞Extensions ＞Haskell Extension Pack を選び install する
VS Code Editor の中で -- または｛- -｝のコメントで
の後ろに何か code を書く
するとHLSが Eval... とプロンプトして来るので　それをクリックする
次の行に Eval code の結果が表示される
例えば次のよう:

-- >>> "Ab"++"cd"
-- "Abcd"

{-
>>> "Ab"++"cd"
"Abcd"
-}
