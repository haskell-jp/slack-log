base changelog 雑まとめ
- 4.9.0
    - モジュールを `semigroups` から移植
        - `Data.List.NonEmpty`
        - `Data.Semigroup`
    - モジュールを `transformers` から移植
        - `Data.Functor.Classes`
        - `Data.Functor.Compose`
        - `Data.Functor.Product`
        - `Data.Functor.Sum`
        - `Control.Monad.IO.Class`
    - 新モジュール: `Control.Monad.Fail`
    - `Data.Functor.Const` に `Const` を移動
    - 大量にインスタンスを追加
    - `[Char]` の `IsString` インスタンスの曖昧性を修正
- 4.10.0
    - モジュールを `bifunctors` から移動
        - `Data.Bifoldable`
        - `Data.Bitraversable`
    - `Data.Either`: `fromLeft` と `fromRight` を追加
    - `liftA2` is now a method of the `Applicative` class
    - 型レベルプログラミング系:
        - 新モジュール: `GHC.TypeNats`
        - `type family AppendSymbol (m :: Symbol) (n :: Symbol) :: Symbol`
        - `(:~~:)`
    - `Type.Reflection`
- 4.11.0
    - `GHC.TypeLits`: `Div`, `Mod`, `Log2`
    - `Alternative`: `ZipList`
    - `Data.Ord.Down`: インスタンス追加
    - Make `Semigroup` a superclass of `Monoid`
    - `Data.List.NonEmpty.!!`
    - Add `<&>` operator to `Data.Functor`
- 4.12.0
    - STM ??
    - `contravariant` から `Data.Functor.Contravariant` を輸入
    - `haddock` から `GHC.ResponseFile`
    - `Data.Monoid.Alt`
    - `Data.Monoid.Ap`
    - `Data.Ord.Down`
