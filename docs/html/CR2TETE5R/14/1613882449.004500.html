• docker networkを設定する
• --linkを使う(レガシー)

「ghcなどビルドツール」と「postgresql server」両方を含んだ大きな「ビルド用Image」を作り、その中で「db 起動 -> db schema migration -> servant-appのビルド」を行うという手法もあります。
