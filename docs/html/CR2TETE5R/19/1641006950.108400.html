何はともあれ GHCup でしょうか。 パッケージ管理にStackを使うのであれ、cabal-installを使うのであれ、とりあえず

https://www.haskell.org/ghcup/

の手順に従って GHCupをインストールすれば、そのほかのツール (GHC, cabal-install, Stack, HLS) は

$ ghcup install <tool-name> <version-specifier>

で自由にインストールできるので。
