f n = if n == 0 then 0 else f (n - 1) + 1 このような関数は、`+ 1` を計算するためにスタックを消費するのでスタックオーバーフローします。`f n = if n == 0 then 0 else f (n - 1)`` の場合はその必要がないのでオーバーフローを避けられます(末尾再帰)
