:{ / :} コマンドを使うと複数行の定義を書けます。使用例は
Prelude> :{
Prelude| hoge 0 = 0
Prelude| hoge 1 = 1
Prelude| hoge x = hoge (x-1) + hoge (x-2)
Prelude| :}

という感じです。
