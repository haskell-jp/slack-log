igrep
御社でご用意できるなら、
チラシ + ノベルティー + スポンサーセッション
全部OKとします。
「お金を払ったら○○しないといけない」というルールではなく「お金を払ってできる・かつやる気があれば」というポリシーでやっていただこうと思います。

ただ、スポンサーセッションについては時間と会の雰囲気の兼ね合いがあるのでみんなが手を挙げた場合はちょっと調整させていただきます（今のところスケジュールは空き気味なので短いものであれば十分可能かとは思いますが）

協賛金の金額による差異は、
- イベントページでの掲載順
- 当日の司会による紹介順
- 当日の会場におけるチラシ・ノベルティーの配置順
によって区別させていただく方向で検討しています。
いずれも同額の場合は会社名のあいうえお順で。
