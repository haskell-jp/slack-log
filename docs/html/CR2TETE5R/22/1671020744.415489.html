GHCiに直接打つ場合は、 :{ と :} で囲うとうまく行きます:

:{
qsort[]=[]
qsort(x:xs)=qsort smaller ++[x]++qsort larger
 where
  smaller = [a | a<- xs,a<=x]
  larger  = [b | b<- xs,b>x]
:}

なお、 x;xs ではなく x:xs と、セミコロンではなくコロンを使う点にご注意ください。
