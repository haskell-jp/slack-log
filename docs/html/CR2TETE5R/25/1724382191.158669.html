import Control.Monad (join)

Hoogle で join を調べ以下のような定義ということがわかりました。
join              :: (Monad m) => m (m a) -> m a
join x            =  x >>= id

これらをまとめると、以下になると思います。
join (*) = (*) >>= id

しばらく考えて、なんとなく関数モナドかなと思い、定義に当てはめて
h >>= f = \w -> id ((*) w) w

ここに引数を与えて、最終的に
id ((*) 5) 5 ==> (*) 5 5

と、ここまできてようやく納得できました。
