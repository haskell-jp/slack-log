f : X→Y
f(0_X)=0_Y
f(1_X)=1_Y
f(a+b)=f(a)+f(b)
f(a・b)=f(a)・f(b)
がほぼ線型写像の性質そのもので、素体は線型ベクトル空間としての基底が1つしかないのでfの行列表現がA=(1)しかなくなるという風に理解しています
