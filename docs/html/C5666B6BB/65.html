例えば， len :: Int の時，
last $ take len $ cycle [1..100::Int]

は cycle=cycle2 ならば二つの Int# に関する単純なループに展開されます． cycle=Data.List.cycle ならば take len が cycle で出てきたリストを食いつぶす（consして直ちにパターンマッチで消費する） ヒープ上に存在し続ける cycle [1..100] を舐め回すコードになりますね．
